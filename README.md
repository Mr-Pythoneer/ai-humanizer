# AI Humanizer

🔗 **Live site:** https://mr-pythoneer.github.io/ai-humanizer/

Paste AI-generated text in, get text back with common AI tells removed — stock vocabulary, transition words, hedging phrases, ethical disclaimers, low burstiness — plus a checklist for the things no script can fix for you.

## Transforms

- AI vocab swap (*delve, nuanced, multifaceted, leverage, ...*)
- Transition-word purge (*furthermore, moreover, in conclusion, is a testament to, ...*)
- Hedging-phrase removal (*it is worth noting that, one might argue, ...*)
- First-person softening
- Contractions
- Ethical-disclaimer removal
- Burstiness (sentence-length variance, em-dashes, paragraph asymmetry)
- Informal voice injection
- List breaker

Everything runs client-side in the page — no text is sent anywhere.
